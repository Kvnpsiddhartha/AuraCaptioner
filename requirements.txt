# Prompt 3C (src/audio.py) - faster-whisper and webrtcvad are intentionally
# NOT installed in the evaluation image. Both are imported lazily and
# guarded: if unavailable, has_speech() returns False and transcribe()
# returns None. For Track 2 clips (nature/urban/sports/cooking) there is no
# meaningful speech content that benefits captioning — skipping saves the
# heavy CTranslate2 binary (~80 MB wheel) and eliminates the runtime
# HuggingFace model download that would otherwise happen on first call.
# To re-enable: pip install faster-whisper>=1.0 webrtcvad>=2.0.10

# Prompt 3D (src/ocr.py) - opencv-python-headless and rapidocr-onnxruntime
# are intentionally NOT installed in the evaluation image. opencv is used for
# the MSER text-region gate; rapidocr is the heavy OCR engine. Both are
# imported lazily and guarded: extract_text() returns [] on any import failure.
# Track 2 clips do not require on-screen text extraction for accurate
# VLM-grounded captions. Removing these saves ~100 MB of native binaries and
# eliminates ONNX Runtime cold-start overhead.
# To re-enable: pip install opencv-python-headless>=4.9 rapidocr-onnxruntime>=1.3
